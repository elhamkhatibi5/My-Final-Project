# Pocket Classroom — Offline

A simple single-page app (HTML/CSS/Vanilla JS) for authoring small learning "capsules" (notes, flashcards, quiz),
studying them in Learn mode, and exporting/importing JSON.

## How to use

Open `index.html` in a modern browser (Chrome/Edge/Firefox). Storage uses LocalStorage (offline).
Use "New Capsule" to author, or import the sample JSON in `exports/sample-capsule.json`.

Demo video (provided by student): https://youtu.be/2oIejLyD_Ro

## Structure

- index.html
- css/style.css
- js/*.js
- exports/sample-capsule.json
